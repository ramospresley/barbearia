Sistema completo de agendamento para barbearias desenvolvido em PHP, com autenticação em dois fatores via SMS e painel administrativo. Projeto desenvolvido para o Projeto Integrador I da UNIVESP.

---

Sobre o Projeto

Este sistema foi desenvolvido como parte do **Projeto Integrador I** da **UNIVESP** (Universidade Virtual do Estado de São Paulo), para o **eixo de Computação**. O objetivo é criar uma solução profissional para gestão de agendamentos em barbearias, com foco em usabilidade, segurança e experiência do usuário.

Objetivos Alcançados

- ✅ Sistema completo de agendamento online
- ✅ Autenticação segura com 2FA via SMS
- ✅ Painel administrativo intuitivo
- ✅ Gerenciamento de horários e serviços
- ✅ Design responsivo e profissional

Funcionalidades

Para Clientes
- Visualização de calendário com horários disponíveis
- Seleção de múltiplos serviços (corte, barba, procedimentos químicos)
- Combos personalizados com desconto
- Login com primeiro nome e telefone + código SMS
- Confirmação via SMS em tempo real
- Cancelamento de agendamentos

Para Barbeiros (Admin)
- Dashboard com visão geral dos agendamentos
- Confirmação de agendamentos
- Cancelamento de agendamentos
- Bloqueio de horários específicos
- Inclusão de novos serviços
- Envio manual de lembretes SMS
- Estatísticas (agendamentos hoje, confirmados, faturamento)
